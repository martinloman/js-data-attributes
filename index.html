Demo av data-attribut i HTML

Öppna konsolen, klicka på knapparna och se vad som skrivs ut. Titta sedan på koden i app.js för att se hur den är skriven.

Titta i style.css för att se hur data-attribut, och andra attribut, kan användas för styling.

Mer info om data-attribut på MDN web docs och YouTube.
